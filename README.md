Repositorio del Curso básico de DevOps dictrado por el equipo de [Craftech](https://craftech.io).

El curso consistió en 4 clases teórico-prácticas y un desafío final:

1. [Docker](https://github.com/tomasspi/curso-basico-devops-marzo-2021/tree/main/1-docker): se dieron los conceptos básicos, los comandos y el desarrollo de imágenes (*Dockerfiles*) utilizando una simple aplicación web diseñada en Flask.
2. [Docker-compose](https://github.com/tomasspi/curso-basico-devops-marzo-2021/tree/main/2-docker-compose): se mostraron la forma de documentar y configurar todas las dependencias de servicio de la aplicación web, agregando un contador de visitar en Redis.
3. [Kubernetes](https://github.com/tomasspi/curso-basico-devops-marzo-2021/tree/main/3-kubernetes): se dictaron los conceptos básicos tanto de la tecnología como de sus componentes y se realizó el despliegue y control de la webapp.
4. [*CI/CD*](https://github.com/tomasspi/curso-basico-devops-marzo-2021/tree/main/4-ci-cd): se enseñó cómo diseñar un *pipeline* de integración continua (CI) y entrega continua (CD) utilizando *Github actions*.

El [desafío](https://github.com/tomasspi/curso-basico-devops-marzo-2021/tree/main/5-desafío) consistió en aplicar todo lo visto en esas clases.
